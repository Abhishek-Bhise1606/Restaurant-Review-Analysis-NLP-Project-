# 🍽️ Restaurant Review Analysis

## 📌 Project Overview

This project focuses on analyzing restaurant reviews using **Natural Language Processing (NLP)** and **Machine Learning** techniques. The goal is to classify customer reviews as **positive or negative** and extract meaningful insights from textual data.

---

## 🎯 Objectives

* Perform text preprocessing and cleaning
* Convert text data into numerical format
* Train a machine learning model for sentiment classification
* Evaluate model performance

---

## 📂 Dataset

The dataset consists of restaurant reviews with corresponding sentiment labels:

* **Review** → Text data
* **Sentiment** → Positive / Negative

---

## ⚙️ Technologies Used

* Python
* Pandas & NumPy
* Matplotlib / Seaborn
* Scikit-learn
* Natural Language Toolkit (NLTK)

---

## 🔄 Workflow

### 1. Data Preprocessing

* Remove punctuation and special characters
* Convert text to lowercase
* Remove stopwords
* Apply stemming

### 2. Feature Extraction

* Bag of Words (CountVectorizer)
* Convert text into numerical vectors

### 3. Model Building

* Train using classification algorithms (e.g., Naive Bayes / Logistic Regression)

### 4. Model Evaluation

* Accuracy Score
* Confusion Matrix

---

## 📊 Results

* The model successfully classifies reviews into positive and negative categories
* Achieves good accuracy on test data
* Helps in understanding customer sentiment
